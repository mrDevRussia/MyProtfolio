# Modern Portfolio Website

A responsive and interactive portfolio website for Full-Stack Web Developers. This portfolio features a modern design with a purple color scheme, smooth animations, and is ready to be deployed on GitHub Pages.

## Features

- Responsive design that works on all devices
- Modern UI with purple color scheme
- Interactive elements and animations
- Project filtering by category
- Contact form with validation
- Smooth scrolling navigation
- Social media integration
- Skills showcase section
- About me section with statistics

## Technologies Used

- HTML5
- CSS3 (with CSS variables and flexbox/grid layouts)
- JavaScript (vanilla, no frameworks)
- Font Awesome for icons
- Google Fonts

## How to Use

1. Clone or download this repository
2. Customize the content in `index.html` to match your information:
   - Update your name, title, and description
   - Replace the placeholder text in the About section
   - Update your skills in the Skills section
   - Add your own projects to the Projects section
   - Update your contact information
3. Replace the placeholder images:
   - Add your profile picture as `assets/profile.png`
   - Add project images as `assets/project1.jpg`, `assets/project2.jpg`, etc.
   - Add your resume as `assets/resume.pdf`
4. Customize colors in the CSS if desired (edit the variables in `:root` in `styles.css`)
5. Update your social media links

## Deploying to GitHub Pages

1. Create a new repository on GitHub
2. Push your code to the repository
3. Go to the repository settings
4. Scroll down to the GitHub Pages section
5. Select the branch you want to deploy (usually `main` or `master`)
6. Click Save
7. Your site will be published at `https://yourusername.github.io/repository-name/`

## Creating the Required Assets

Before deploying, make sure to create an `assets` folder and add the following files:

```
assets/
├── profile.png       # Your profile picture (recommended size: 500x500px)
├── project1.jpg      # Project image (recommended size: 600x400px)
├── project2.jpg      # Project image (recommended size: 600x400px)
├── project3.jpg      # Project image (recommended size: 600x400px)
├── project4.jpg      # Project image (recommended size: 600x400px)
└── resume.pdf        # Your resume in PDF format
```

## Customization Tips

- **Colors**: Edit the CSS variables in the `:root` selector in `styles.css` to change the color scheme
- **Fonts**: Replace the Google Fonts link in the `<head>` of `index.html` to use different fonts
- **Projects**: Add or remove project cards in the Projects section of `index.html`
- **Skills**: Modify the skill items in the Skills section to match your expertise
